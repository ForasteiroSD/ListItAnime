Este é um aplicativo feito na web que permite aos usuários salvar seus animes assistidos e animes para assistir, enquanto cria uma lista dos animes melhores avaliados pelo usuário. O aplicativo foi criado com o intuitivo de ajudar seus criadores a avaliar seus animes assistidos e como o projeto final do curso Programação Web.

As tecnologias utilizadas foram:
    Frontend: React com Vite
    Backend: NodeJs com Express
    Packages: Alguns packages utilizados foram: Axios, React Router, Zod, Js-Cookie, Js-Sha256, Prisma e Cors.

O projeto tenta resolver um problema de organização dos animes assistidos e que deseja assistir, além de obter uma lista de seus animes favoritos.

O problema resolvido pela aplicação é importante pois os criadores desejavam algo que tivesse as funcionalidades implementadas e, portanto, irão utilizá-la.
